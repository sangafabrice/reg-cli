# Set-ChromiumShortcut

## SYNOPSIS
Save a specified chromium application shortcut to Start Menu.

## SYNTAX

```
Set-ChromiumShortcut [-Path] <String> [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### Example 1
```powershell
PS C:\> Set-ChromiumShortcut "C:\Secure\AvastBrowser.exe"
```

## PARAMETERS
